# World cup 2026 random project

  - Step 1 — Pick a match: Browse 72 World Cup fixtures ( group match ) with a search filter. Click any card to select it.
  - Step 2 — Add players: Type names and press Enter (or click Add). Each player shows in an editable list — you can rename or remove anyone. A running count shows the expected split (e.g. 22 → 11 vs 11).
  - Step 3 — Results: Players are randomly shuffled and split as evenly as possible. Hit "Generate again" to re-randomize the same group instantly.

32 matches will be updated later, after 72 matches.
